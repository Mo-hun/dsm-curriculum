# 2019 머신러닝 로드맵

(2019.6.4 수정)

<br/>

## 개요

고등학교 3학년동안 딥러닝을 완전히 정복하기에는 어려움이 있습니다. 그렇기 때문에 이 커리큘럼에서는 딥러닝에 필요한 기초적인 수학만을 포함하고 있습니다. 그리고 머신러닝의 한 분야인 딥러닝만을 위주로 다룹니다. 개발 언어는 빠르고 쉽게 학습할 수 있는 파이썬을 이용한 자료를 주로 안내합니다.
